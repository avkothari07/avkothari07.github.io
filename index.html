Mach5 Interceptor

Description

Mach5 is database platform start-up that will have the same capabilities and functionalities as Elasticsearch with additional features. The Interceptor project's goal was to create a Rust program that proves that the new Mach5 database and Elasticsearch have the same functionality. As a part of the project, I built reverse proxies with Nginx and Rust to log requests and responses between Kibana and Elasticsearch. I also designed and wrote a program to send and replay HTTP requests to Elasticsearch and new Mach5 database to illustrate shared functionality in Rust.

Note

Because the Interceptor is a key piece of Mach5's new technology, I cannot share source code for this project; however, please feel free to reach out to me with any questions or comments about the project.

StudyPay Application

Description

StudyPay is an app utilized by the Nock Lab at Harvard University to track and compensate research participants for completing forms and other tasks for research projects. In the summer of 2019, I redesigned this application with the lab's lead software engineer.
